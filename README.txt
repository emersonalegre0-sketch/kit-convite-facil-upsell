PÁGINA DE UPSELL — KIT CONVITE FÁCIL

1. Publique a pasta inteira em uma hospedagem estática.
2. Use index.html como página inicial.
3. Não remova o script oficial upsell.js.
4. O código está configurado com a oferta Cakto: 38iktg5.
5. Aceitar e rejeitar direcionam para members_area.
6. Depois de publicar, cole a URL completa desta página no campo de upsell da Cakto.
7. Faça uma compra de teste antes de iniciar os anúncios.

Observação: fora do fluxo real de compra da Cakto, o botão pode não concluir uma cobrança. A validação final deve ocorrer através de uma compra aprovada de teste.
